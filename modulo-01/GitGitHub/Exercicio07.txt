
# 🧪 Exercício 7 – Badges do GitHub

---

## 🎯 Objetivo

Aprender a utilizar **Badges (selos)** no `README.md` para deixar o projeto mais profissional e visualmente organizado.

---

## 🏷️ O que são Badges?

Badges são pequenos selos informativos que aparecem no topo do README e mostram informações rápidas sobre o projeto, como:

- Linguagem utilizada
- Status do projeto
- Licença
- Versão
